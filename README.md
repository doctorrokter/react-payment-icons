# React Payment Icons

Library to get payment method icons

## How to use

Using this component on your project.

```jsx
  import PaymentIcon from 'react-payment-icons';

  <PaymentIcon
    id="visa"
    style={{ margin: 10, width: 100 }}
    className="payment-icon"
  />
```
